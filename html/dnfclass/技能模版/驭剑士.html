剑术精通[回到顶部]

学习后， 可以使用驭剑士专属的基础攻击动作






血影之太刀精通[回到顶部]

- [附加效果] -
[帝国剑术] : 增加太刀攻击力
[魔剑降临] : 增加攻击力
[驭剑术] : 强制中断技能时， 可增加技能攻击力并减少相应技能的冷却时间； 使用蓄气类技能时 ([恶即斩]、 [破军旋舞斩])， 可直接发动最高阶段蓄气效果； 增加强制中断次数， 减少恢复时间； 10连击达成时可以恢复1次[驭剑术]
[瞬影三绝斩] : 强制中断[驭剑术]时， 追加发射剑气






波动之钝器精通[回到顶部]

使用钝器系武器攻击敌人时， 增加钝器的物理攻击力， 且附加特殊攻击效果
- [附加效果] -
[跃空斩] : 无需在跳跃状态下施放
[穿云破空剑] : 增加冲击波范围和攻击力
[幻剑术] : 攻击时会产生冲击波
[雷鸣千军破] : 只发动最后一击， 且会产生冲击波； 多段爆炸变为单次爆炸； 减少气剑生成范围， 增加攻击力
冲击波命中敌人时， 增加技能攻击力， 效果持续一定时间






魔性之短剑精通[回到顶部]

使用短剑系武器攻击敌人时， 增加短剑物理攻击力， 且附加特殊攻击效果
[附加效果]
魔剑降临 : 可以重叠[魔剑降临]， 同时拥有2把魔剑效果， 且减少冷却时间； 重叠[魔剑降临]时增加技能攻击力； 最后幻化的魔剑为主魔剑， 前一个为副魔剑； 副魔剑的攻击力为基础攻击力的一定百分比
魔剑降临 : 减少冷却时间
魔剑奥义 : 增加攻击力、 减少冷却时间
瞬影三绝斩 : 增加异常状态几率和攻击力






毁灭之巨剑精通[回到顶部]

使用巨剑系武器攻击敌人时， 增加巨剑物理攻击力， 且附加特殊攻击效果
- [附加效果] -
[连环斩] : 施放时有霸体判定
[瞬影三绝斩] : 施放时有霸体判定
[破军旋舞斩] : 施放时有霸体判定； 多段攻击变更为一次攻击
[破军旋舞斩]、 [恶即斩]、 [帝国剑术]巨剑蓄气时， 增加该技能攻击范围和所有技能的攻击力
[破军旋舞斩] : 巨剑蓄气时增加攻击力
[恶即斩] : 增加蓄气时间， 巨剑蓄气时增加最终一击攻击力






魔剑降临[回到顶部]

汇聚魔力， 将佩戴的武器任意幻化成4把魔剑中的一把， 每把剑都有固定属性
魔剑包括 : 火焰之普朗兹(火)、 寒冰之凯拉丁(冰)、 闪电之斯灵格(光)、 冥炎之巴萨达(暗)
施放技能后， 可以利用方向键选择所需的魔剑； 若按下空格键， 可以解除[魔剑降临]； 普通攻击或施放[浮空击]时， 有一定几率触发各自的属性效果
掌握[魔性之短剑精通]后， 可以再幻化一把不同属性的魔剑







驭剑术[回到顶部]

加强自我控制力； 可强制中断当前进行的技能攻击动作， 并立即施放其它技能
强制中断效果有次数限制， 达到上限后， 需要经过一定时间才能恢复使用
[学习剑心 : 魔剑唤醒后]
增加可以强制中断的次数






自动招架[回到顶部]

有一定的几率以剑士的本能进行招架， 效果持续一定时间






人剑合一[回到顶部]

将武器与身心合一， 探知敌人的弱点， 进行致命的攻击
该效果持续时间内， 可提高自身暴击伤害和暴击率







魔剑奥义[回到顶部]

除普通攻击、 [浮空击]以及Buff类技能之外， 使用其它技能攻击敌人时， 有一定几率触发魔剑的属性效果
使用暗属性的[冥炎之巴萨达]时， 可以召唤冥炎进行范围攻击； 使用冰属性的[寒冰之凯拉丁]时， 可以施放寒冰之枪进行攻击， 并有一定几率使敌人进入冰冻状态； 使用火属性的[火焰之普朗兹]时， 可以施放燃烧之眼进行攻击， 并有一定几率使敌人进入灼伤状态； 使用光属性的[闪电之斯灵格]时， 可以召唤落雷进行攻击， 并有一定几率使敌人进入失明状态
在[攻击类型转换]技能目录中， 可以on/off转换属性的魔法剑的外形。 外形off时， 魔剑的力量隐藏于剑中， 恢复原本的武器攻击范围







魔剑觉醒[回到顶部]

魔剑开始觉醒自我意识， 对敌人发动致命攻击。 增加基本攻击力和转职技能的攻击力






帝国剑术[回到顶部]

只传授给帝国剑士的秘密剑术
根据佩戴的武器不同基本攻击的招式会发生变化
可以结合基本攻击使用， 但施放最后一击后无法衔接基本攻击或[帝国剑术]
根据佩戴的武器不同， 会施放不同的帝国剑术
可使用[驭剑术]强制中断[帝国剑术]并施放其他技能







招架反击[回到顶部]

[招架]成功后可激活该技能； 使用后， 可以进行2次快速斩击






疾影斩[回到顶部]

以闪电般的速度冲向敌人并进行斩击
被击中的目标将进入短暂的僵直状态， 然后倒地
冲刺过程中， 若再次按下技能键， 则会停止冲刺并原地施放斩击







幻剑术[回到顶部]

凝聚出自身的幻影， 并令其攻击前方敌人
幻影在攻击一定次数后会消失
掌握[波动之钝器精通]后， 攻击时会产生冲击波







升龙剑[回到顶部]

向前突进将敌人撞退后， 再用强力上斩攻击， 使敌人浮空
[学习剑心 : 魔剑唤醒后]
进行强力上斩后， 进行快速向下劈击攻击







穿云破空剑[回到顶部]

使用强力的上斩攻击挑飞敌人后， 跃向空中对其进行多段斩击， 最后将敌人砸向地面， 并引发冲击波
对于无法抓取的敌人， 直接施放多段斩击
[学习剑心 : 魔剑唤醒后]
不再击飞敌人， 直接在地面上进行乱舞攻击。最后一击将掉落魔剑降临幻化的武器







破军旋舞斩[回到顶部]

身体旋转一周的同时， 对敌人进行拔刀斩击
蓄气后施放可以增加前冲距离
使用方向键可以控制前冲方向(原地、 前方、 对角线)。 若不指定方向， 则默认向前移动
若已掌握[血影之太刀精通]， 使用[驭剑术]施放该技能时， 则直接施放已蓄满状态的[破军旋舞斩]







瞬影三绝斩[回到顶部]

聚集力量， 向前方连续斩出3道巨大的剑气
根据当前[魔剑降临]的状态， 攻击时， 有一定几率触发对应的属性效果
在决斗场中， 武器的攻击属性不影响技能的攻击属性







雷鸣千军破[回到顶部]

聚集手上的魔力， 召唤出远古巨剑攻击敌人
该技能共进行3次斩击； 最后一次斩击时， 会从地面召唤出多把巨剑并引发强烈爆炸
掌握[波动之钝器精通]后， 攻击时， 只发动最后一击， 且会产生冲击波和爆炸







恶即斩[回到顶部]

向前冲刺一段距离后， 对敌人进行快速的连续斩击
根据蓄气时间的不同， 斩击次数和冲刺距离会有所变化
蓄气到第1~2阶段时， 会发动快速的终结技； 蓄气到第3阶段时， 则会发动强威力的终结技
按向后方向键， 则在原地直接施放
若施放后连续按攻击键， 则增加斩击次数
掌握[血影之太刀精通]后， 利用[驭剑术]的效果发动技能时， 直接发动最大蓄气效果







斩魂术[回到顶部]

利用暴击命中带有‘魂’标志的敌人时， 可以获得增加基本攻击和技能伤害的Buff
Buff持续时间内， 再次利用暴击命中时将会刷新Buff时间






极·驭剑术（时空斩）[回到顶部]

将驭剑术修炼到极致后， 连时空都可以斩断
该技能达到Lv3、 Lv6、 Lv9时， 效果会有所增强
Lv3 : 施放时进入无敌状态
Lv6 : 提升[驭剑术]的技能Lv， 效果持续一定时间
Lv9 : 施放技能后， 一定时间内所有技能的冷却时间减少







烈刃天冲[回到顶部]

前冲攻击前方敌人， 召唤四种属性剑合并后引发爆炸
输入前进键可以向更远处前冲
剑宗召唤一柄魔法剑射向前方， 若击中敌人， 则会生成四种属性剑刺向目标， 最后融合四属性之力形成远古巨剑冲向敌人并抛向地面， 造成爆炸伤害







极·驭剑术（幻剑阵）[回到顶部]

在前方召唤四种属性剑， 并生成幻剑阵压制敌人， 随后召唤远古巨剑给予敌人终极一击
生成幻剑阵时， 会使阵内敌人浮空并滞留， 同时剑宗会跃起对滞空的敌人进行连斩攻击； 发动终极一击时， 则会将敌人轰落至地面， 同时引发强烈的爆炸
施放技能后， 若连续按下攻击键， 可以加快攻击速度







帝皇盟约[回到顶部]

领悟到剑之奥义的剑皇， 更加熟练的使用武器。 依据不同的武器特性， 增加不同程度的物理攻击力






飓风魔剑[回到顶部]

在魔剑注入力量， 生成强力的魔力飓风， 然后射向前方进行多段攻击并引发爆炸
施放时， 可以蓄气控制出招时机
学习该技能后， 可将[魔剑奥义]的力量存储为魔剑堆， 根据[魔剑奥义]的冷却时间， 每隔一段时间可以存储一次魔剑堆
施放技能时， 可以利用存储的魔剑堆强化[飓风魔剑]的技能效果， 最终爆炸攻击力随[魔剑奥义]的攻击力增加而增加
魔力飓风的属性随[魔剑降临]幻化的武器类型变化， 命中敌人时可以引发异常状态， 异常状态Lv与[魔剑奥义]的异常状态Lv相同
只有在[魔剑降临]状态下才能施放该技能







极·驭剑术（破剑阵）[回到顶部]

召唤出远古魔剑的墓地， 然后快速移动攻击敌人， 每次攻击时会引发强烈的爆炸
施放技能后， 若连续按下攻击键， 可加快攻击速度
施放技能时， 若用[驭剑术]强制中断技能， 则所有魔剑同时爆炸







誓约之剑：雷沃丁[回到顶部]

凝缩魔剑的力量， 召唤出凝聚四大元素力量的誓约之剑， 斩破次元的界限
次元震动的期间， 敌人将进入长时间的僵直状态， 当次元破裂时受到巨大的伤害







剑心 : 魔剑唤醒[回到顶部]

利用强大的魔力唤醒魔剑和雷沃汀的真正面貌， 掌控新的力量。 学习后， 增加转职技能攻击力， 部分技能赋予额外效果
[升龙剑] : 上挑敌人后快速向地面下劈
[穿云破空剑] : 变更为地面乱舞攻击， 最后掉落[魔剑降临]选择的魔剑并造成爆炸攻击
[驭剑术] ： 增加可以强制中断的次数







极诣 · 驭剑术 : 聚魔剑[回到顶部]

利用无限魔力的力量， 召唤更加强大的魔剑和雷沃汀冲向前方刺击敌人。 被强化版雷沃汀命中的敌人暂时进入僵直状态， 然后受到强化版魔剑的攻击。 当收起雷沃汀时， 无限的魔力瞬间爆炸
输入方向键时， 可增加前冲距离







誓约之引 : 万剑之巅[回到顶部]

召唤隐藏在次元中的魔剑军团。 魔剑军团在誓约之剑 : 雷沃汀的指挥下， 向敌人斩击后腾空， 随后冲向地面歼灭敌人
