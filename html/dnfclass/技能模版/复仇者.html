恶魔之力[回到顶部]

复仇者特有技能； 可以使基本攻击产生变化， 在进行普通攻击或技能攻击时， 可以积攒恶魔能量。 施放技能后， 可以按Z、 X、 C键， 消耗恶魔能量并用恶魔残影给予敌人暗属性魔法伤害。 普通攻击、 [末日浩劫]、 [地狱之门]、 [不朽战吼]、 [毁灭强击]、 [永堕 : 混沌弑神]以及[魔化 : 末日审判者]状态下无法追加施放[恶魔之力]， 且部分技能不会获得恶魔能量
在决斗场中， 决斗开始时恶魔能量初始值为0







半魔化[回到顶部]

解放部分恶魔之力， 在人形或半魔化状态时， 增加部分技能的攻击力。 施放[半魔化]后， 会进入半人半魔的状态， 增加攻击速度和移动速度， 并施放更为强力的恶魔残影
- 半魔化状态下增加攻击力的技能 :
[恶魔之手]、 [死亡切割]、 [裂地锤]、 [回旋飞镰]、 [复仇之刺]、 [厄运之轮]、 [恶魔之握]、 [恶魔之拳]、 [黑暗权能]、 [地狱之门]、 [末日浩劫]、 [不朽战吼]、 [毁灭强击]及[永堕 : 混沌弑神]
[天地不仁以万物为刍狗！]







镰刀精通[回到顶部]

使用镰刀系列武器攻击敌人时， 基本攻击类型会变成魔法攻击， 同时增加武器的魔法攻击力、 攻击速度、 魔法暴击率和暗属性抗性， 并减少光属性抗性







堕落之魂[回到顶部]

消耗自身的恶魔能量， 在一段时间内增加复仇者的基本攻击力和技能攻击力
[堕落之魂]的技能Lv会随着[恶魔之力]的技能Lv增加而增加







恶魔诅咒[回到顶部]

学习后， 增加复仇者基本攻击和转职技能攻击力。 ([魔化 : 末日审判者]普通攻击、 [恶魔之爪]除外)
使用除[魔化 : 末日审判者]之外的复仇者转职技能时， 发动[恶魔诅咒]Buff， 并增加移动速度






死亡切割[回到顶部]

将恶魔的气息注入到镰刀之中， 并快速挥动镰刀给予敌人多段暗属性魔法伤害； 每次攻击可恢复一定量的HP。 若连续按技能键， 则可以加快挥动速度
若在[魔化 : 末日审判者]状态下施放该技能， 则增加攻击力和冷却时间







裂地锤[回到顶部]

将恶魔的气息聚集到拳头后， 用力锤击地面击起乱石。 乱石柱从地面出来后， 对周围敌人造成暗属性魔法伤害
若在[魔化 : 末日审判者]状态下施放该技能， 石柱变为黑暗石柱， 并增加乱石的数量和冷却时间
黑暗石柱会追加爆炸， 对敌人造成伤害







回旋飞镰[回到顶部]

向前方投掷回旋的恶魔镰刀， 使敌人受到暗属性魔法伤害。 若再按一次技能键或待一定时间后可收回镰刀， 同时给予敌人多段暗属性魔法伤害
若在[魔化 : 末日审判者]状态下施放该技能， 则增加攻击力和冷却时间







复仇之刺[回到顶部]

将身体内的恶魔气息释放出来， 形成复仇之刺， 可以给敌人造成暗属性魔法伤害， 并使其进入僵直状态
若在[魔化 : 末日审判者]状态下施放该技能， 则会追加打击效果， 并增加冷却时间
被击状态下也能施放技能， 被击状态下施放技能时， 攻击力会增加







厄运之轮[回到顶部]

利用恶魔的力量将自身变为巨大的齿轮， 向前高速旋转并攻击敌人， 使其受到多段暗属性魔法伤害。 可以在跳跃状态下使用， 按方向键可以改变齿轮的攻击方向
若在[魔化 : 末日审判者]状态下施放该技能， 会进化为死亡打击， 并增加攻击力和冷却时间
死亡打击会瞬间斩过敌人， 并造成伤害







恶魔之握[回到顶部]

借助恶魔之手攻击敌人。 若敌人被恶魔之手握住， 则进入僵直状态， 并会被拖向复仇者。 在魔化状态下， 会出现更巨大的恶魔之手







恶魔之拳[回到顶部]

从自己的影子中抽取恶魔的气息缠绕在手臂上， 然后向前方扩散
对敌人进行一定次数的多段攻击并短时间强制控制敌人， 然后爆炸， 对周围敌人造成伤害
在决斗场中， 只有攻击之前的动作为霸体状态， 爆炸范围内的敌人会被拉到复仇者的位置







黑暗权能[回到顶部]

利用恶魔的锁链， 将周围的敌人锁在空中， 并召唤出黑暗晶体给予敌人多段暗属性魔法伤害。 对于不可抓取的敌人直接利用黑暗晶体进行攻击
若在[黑暗权能]施放时， 再次激活技能键， 则复仇者可以自由移动
若在[魔化 : 末日审判者]状态下施放该技能， 则追加强力的最终一击







恶魔唤醒[回到顶部]

增加恶魔能量、 魔化状态下的基本攻击力、 [恶魔之爪]和[审判]的攻击力
在噩梦的作用下， 增加末日审判者的技能攻击力
[深渊之夜， 极度的孤独已使我的心被冰冷的痛苦之海侵蚀。 噩梦， 也即将向我走来……]






魔化 : 末日审判者[回到顶部]

化为恶魔形态进行战斗。 魔化后恢复恶魔能量， 并在一定时间内进入无敌状态， 同时无法使用部分主动技能。 魔化状态下， 若HP降低为0， 则解除魔化状态
魔化状态下可使用的新技能 : [恶魔之爪]、 [恶魔屏障]、 [审判]
[恶魔之爪]操作指令 : Z
[恶魔屏障]操作指令 : ↓↓ + X
[审判]操作指令 : ←→ + Z
[恶魔屏障]强制攻击倒地敌人时， 若按跳跃键， 可以发动无敌模式， 效果持续一定时间
可以强制中断普通攻击动作， 并立即施放[恶魔之爪]， 也可在蓄气后使用[恶魔之爪]
在空中普通攻击时， 会撕开地面的敌人
Lv3 : 增加移动速度、 攻击速度、 防御力和HP
Lv6 : 魔化后， 增加10%的命中率
Lv9 : 魔化后， 减少10%的技能冷却时间， 效果持续一定时间
[他的内心里还留有披上神圣的祭司服时的纯粹、 正直、 慈爱与怜悯。]







恶魔屏障[回到顶部]

生成一道屏障， 减轻自身受到的伤害
只能在魔化状态下使用
[神的惩罚者从不畏惧痛苦……和死亡！]







审判[回到顶部]

将右臂变为恶魔本体的模样对敌人行刑
按向前键施放技能时， 伸出巨大的手臂抓取敌人并砸向地面， 然后拖动一定距离后抛出。 对于无法抓取的敌人， 则直接握住并粉碎敌人
只有在半魔化及魔化状态下才能聚集恶魔之力施放[审判]。 若在[魔化 : 末日审判者]状态下施放该技能， 则可以增加攻击力







地狱之门[回到顶部]

利用恶魔的力量打开地狱之门， 从中释放出大量的恶魔气息攻击敌人。 在[魔化 : 末日审判者]状态下， 技能攻击力会大幅提升







末日浩劫[回到顶部]

高高跃起， 并在下方形成魔法阵吸附敌人。 经过一段时间或追加操作时， 复仇者会在高空正对魔法阵中心直冲而下， 产生冲击波， 可对敌人造成大量伤害
在[魔化 : 末日审判者]状态下， 复仇者会造成更大的灾难







原罪之力[回到顶部]

恶魔的力量进一步得到进化。 增加[恶魔之力]的攻击力， 并且大幅增加基本攻击和[恶魔之力]之外的所有技能攻击力。 而且， 施放[毁灭强击]、 [不朽战吼]、 [永堕 : 混沌弑神]过程中可以立即魔化







不朽战吼[回到顶部]

发出恶魔的战吼对附近敌人造成大量伤害， 并使周围的敌人进入诅咒和失魂状态。 攻击失魂状态的敌人可以提取他们的灵魂， 汲取灵魂能恢复恶魔能量。 未汲取的灵魂经过一定时间后会消失
在魔化状态下使用时， 技能会变为多段攻击， 对敌人造成更大的伤害
若按住技能键， 则可以蓄气使用， 用消耗恶魔能量来换取施放额外的[不朽战吼]。 在半魔化状态下使用时， 可以减少追加恶魔能量的消耗量
[啊， 没有归宿的可怜的野兽……]







毁灭强击[回到顶部]

使用恶魔之角突进并聚拢敌人， 然后再发动强力的黑暗冲击波对范围内的敌人造成暗属性伤害
被黑暗冲击波命中的敌人会因被暗炎附身而受到持续性伤害。 长按技能键可以蓄力。 蓄力时会消耗恶魔能量， 并且暗炎的击打次数会增加
在魔化状态下， 可以增加技能攻击力； 在变形状态下， 可以减少追加操作时的恶魔能量消耗量
[你以为世人会歌颂你们的功绩吗？ 他们只会在被救赎后， 用轻蔑的眼神指责你们， 用苛责的方式打压你们， 让你们终生悔过和赎罪。 --致永生者]







永堕 : 混沌弑神[回到顶部]

释放恶魔之力撕碎前面的所有敌人。 施展强力的连锁攻击， 每次撕碎敌人时可恢复恶魔能量
施放后连续按x键， 可以增加攻击速度。 按c键， 可以直接进行终结一击
若技能结束之前按x键或z键， 则可以通过消耗额外的恶魔能量， 进行更强力的终结一击。 半魔化状态下减少恶魔能量消耗量
[看着眼前敌人一个个被撕成碎片， 我的意识却在逐渐模糊。 在临晕倒的那一刻， 我知道一切都结束了， 等待自己的唯有死亡……]







光之影[回到顶部]

将恶魔的力量释放到极限，可以使用更强大的恶魔之力，绝不与无止尽的恶魔低语，凭借真挚信念获得驱使光的力量
增加神启·复仇者基本攻击力和转职技能攻击力，并赋予部分[魔化：末日审判者]状态下的技能附加效果
裂地锤
- [魔化 : 末日审判者]状态下施放时， 在前方生成多个黑暗石柱， 并使其爆炸
- 黑暗石柱的生成和爆炸攻击变更为范围攻击判定。黑暗石柱的攻击力和爆炸攻击力随裂地锤攻击力增加而增加
回旋飞镰
- [魔化：末日审判者]状态下施放时，在攻击范围内以攻击次数上限对敌人进行攻击
- 镰刀投掷攻击力和回收攻击力与回旋飞镰攻击力和多段攻击力相同







极恶洪流[回到顶部]

释放恶魔的力量，在头上生成暗黑球
从生成的球体发射光线，横扫前方的敌人
长按技能键消耗恶魔能量，可以凝聚内心的光之力，发射更强大的光线
[魔化：末日审判者]状态下使用时，造成额外伤害
[半魔化]状态下使用时，减少恶魔能量消耗







末日福音 : 毁灭之翼[回到顶部]

将恶魔的力量释放到精神的极限，进入真·恶魔化的状态，但低语愈加明显
真·恶魔化变身后，撕裂自己的翅膀，投向地图中最强大的敌人
投掷的翅膀穿刺敌人并使其悬空，复仇者突袭后拔出刺入的翅膀
拔出翅膀时，在冲击的余波下出现黑暗空间，复仇者将自己的全部力量集中于翅膀之上，形成巨大的光之镰刀，毁灭敌人和黑暗空间
[魔化：末日审判者]状态下使用时，造成额外伤害
施放技能过程中无法使用[恶魔屏障]







极限死亡切割[回到顶部]

将恶魔的气息注入到镰刀之中， 用更快的速度挥动镰刀给予敌人多段暗属性魔法伤害； 每次攻击可恢复一定量的HP。 若连续按技能键， 则可以加快挥动速度







黑暗之触[回到顶部]

从自身的影子中伸出恶魔的爪子攻击，给敌人造成暗属性魔法伤害
