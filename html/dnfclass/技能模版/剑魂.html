肥猫的秘密基地

 


光剑掌握[回到顶部]

可以使用光剑系武器攻击敌人，同时可以增加攻击速度，减少技能冷却时间






武器奥义[回到顶部]

领悟武器奥义， 可增加所有武器的精通Lv
掌握该技能Lv1时， 可使所有武器精通各提升Lv2； 以后技能每提升Lv1时， 武器精通提升Lv1







疾影手[回到顶部]

剑魂特有技能， 可以迅速切换武器， 并且在切换其它系列武器时， 可以增加攻击速度和移动速度， 效果持续一定时间






光剑精通[回到顶部]

使用光剑系武器攻击敌人时，可以增加光剑的物攻、魔攻和命中率






自动格挡[回到顶部]

在自身被击中的前一秒以剑士的本能进行格挡， 且施放自身已学的[格挡]技能
在地下城时， 可以使自身进入霸体状态， 效果持续一定时间






破极兵刃[回到顶部]

使武器突破极限并持续一定时间，可以增加基本攻击力、技能攻击力和物理暴击率






逆转反击[回到顶部]

自身背后遭受攻击时， 可以进行反击； 但作出反击时不会减少伤害






无我剑气[回到顶部]

达到无我境界，增加基本攻击力和技能攻击力






神影手[回到顶部]

掌握疾影手的奥义，切换武器时，可在前方发动冲击波，并获得[疾影手]Buff效果
无法在被击时使用[神影手]，若不符合[神影手]的施放条件时，则会发动[疾影手]







后跳斩[回到顶部]

施放[后跳]的同时向敌人发出斩击







里·鬼剑术[回到顶部]

剑魂特有的鬼剑术，使用的武器不同，出现的攻击效果也不同







流心[回到顶部]

属于剑魂的独有技能
可以强制中断普通攻击、[里·鬼剑术]、[逆转反击]，立即施放[流心]，[流心 : 狂]只能在施放[流心]后才能使用







流心：刺[回到顶部]

在施放[流心]技能状态下，对前方敌人发动强威力的突刺
突刺的附加效果受武器类型影响。使用除钝器以外的武器时，在空中按流心键可以发动空中形态







流心：跃[回到顶部]

在施放[流心]技能状态下，向前轻轻跳跃，在跳跃状态下可通过追加操作向敌人发出砍击；使用方向键可以进行Y轴移动







流心：升[回到顶部]

在施放[流心]技能状态下，向上跳跃的同时对敌人发出上挑攻击







流心：狂[回到顶部]

该技能只能在施放[流心]技能后使用；蓄气后使用[流心:刺]、[流心:跃]、[流心:升]，可以增加这些技能的攻击力和物理暴击率







破军升龙击[回到顶部]

以冲撞击退敌人的同时向敌人发出单手上斩，可以使敌人浮空
若已学一定Lv以上的[巨剑精通]或[钝器精通]并使用巨剑或钝器系武器，则可以追加一个蓄气型下捶击。被钝器下锤击命中的敌人会受到大量的冲击波伤害
使用光剑时，增加冲撞距离和多段攻击次数







拔刀斩[回到顶部]

拔刀并快速向周围敌人发出强威力的斩击，可以攻击大范围的敌人，但准备动作时间稍长
若已学一定Lv以上的[太刀精通]或[光剑精通]并使用太刀或光剑系武器，则可以追加一个拔刀追击
使用太刀时，可以同时叠加多个刺伤效果；[太刀精通]达到Lv10时，可将刺伤引爆
使用钝器时，可以使敌人倒地并对倒地敌人追加伤害







猛龙断空斩[回到顶部]

向前方快速移动并斩击敌人，可以上下左右移动着斩击敌人
起始斩击后可以通过追加操作指定后续的攻击方向，最后一击为可以使敌人浮空的单手上斩
若已学一定Lv以上的[光剑精通]并使用光剑系武器，则可以使敌人进入感电状态







破军斩龙击[回到顶部]

以冲撞击退敌人后使敌人浮空，并使其进入眩晕状态
使敌人浮空后，施放者还会快速转身并用剑刺敌人，且将刺中的敌人聚在一处后发出上挑攻击，使敌人浮空
使用太刀时，可以同时叠加多个刺伤效果；[太刀精通]达到Lv10时，可将刺伤引爆







幻影剑舞[回到顶部]

在原地向敌人发出迅猛的连续斩击，且最后一击可以发出凌厉的剑气并给予敌人多重伤害
斩击次数、速度和最后一击的效果，会因使用的武器以及武器精通技能的不同而出现差异
使用短剑、太刀、巨剑、光剑系武器时，最后一击可以发出凌厉的剑气并给予敌人多重伤害
可以按上下方向键控制剑气的方向，若连续按攻击键，则可以增加斩击次数和剑气威力
习得[短剑精通]后短剑系武器的剑气多段攻击间隔较短，但是持续时间长
钝器系武器的最后一击不会发出剑气，但会生成冲击波
若相应的武器精通技能达到Lv7以上，则使用光剑和太刀系武器时，可以增加斩击次数和攻击速度，但攻击力会减少
若角色佩戴太刀使用该技能时，可以造成额外的刺伤效果。[太刀精通]达到Lv10时，可将刺伤引爆







极·鬼剑术(斩铁式)[回到顶部]

将鬼剑术修炼至极致， 增加基本攻击力和技能攻击力。 而且， 攻击时可以使敌人进入所受暴击伤害增加的状态， 效果持续一定时间






极·鬼剑术(斩钢式)[回到顶部]

学习后，[极·鬼剑术(斩铁式)]的队伍加成效果转变为增加自身技能攻击力






极·鬼剑术(暴风式)[回到顶部]

向空中同时抛出24把剑， 使其中12把浮空， 另外12把插入地面形成剑阵， 并给予敌人高僵直的物理伤害； 施放者用这24把剑攻击周围敌人， 在剑阵中受到伤害的敌人会被吸进剑阵中心。 所有的剑全部消失后， 向敌人发出上斩一击作为终结， 并给予敌人巨大伤害。 在[流心]状态下也可以发动， 此时不强制控制敌人， 直接发动攻击
Lv3、 Lv6、 Lv9时， 技能效果将有所增强
Lv3 : 施放后， 在一段时间内使自身进入无敌状态
Lv6 : 施放时， 增加物理暴击率
Lv9 : [极 · 鬼剑术 (暴风式)]攻击力增加10%







极·神剑术(流星落)[回到顶部]

高高跃起并引发强烈的冲击波， 然后在空中向目标地点投掷数十把流星剑， 最后向下重击敌人
若已掌握[极 · 神剑术]则会进行追加攻击







破空拔刀斩[回到顶部]

拔刀并快速向周围敌人发出强威力的斩击， 可以攻击大范围的敌人， 并向前方敌人发动剑气攻击
通过短暂蓄力剑气的僵直过程后， 通过拔刀施放近距离击打并发射圆形剑气
此时近距离击打和圆形剑气不会同时击中目标
佩戴太刀时， 可对多数敌人附加刺伤效果， 若已掌握Lv10以上的[太刀精通]， 则会引爆所有刺伤效果







极·神剑术[回到顶部]

剑术学习达到极致，可以将其他知名剑士的技能融会贯通
施放流心系列技能时，会发动布万加的[霸体护甲]
施放[后跳斩]时，会出现西岚的剑气
施放[猛龙断空斩]时，会出现巴恩的飓风
施放[破军升龙击]时，会发动阿甘左的暴风式
施放[幻影剑舞]时，会触发索德罗斯的幻影攻击效果
施放[极 · 神剑术 (破空斩)]时，可触发极限之十字斩，极限之十字斩优先锁定HP最高的敌人后，以那个敌人为中心，发动大范围的十字斩击
施放[极 · 神剑术 (流星落)]时，可输入方向键和X键投掷3把流星之天剑，可使用方向键控制流星之天剑的掉落方向， 最多可以强制控制10个敌人







极·神剑术(破空斩)[回到顶部]

以极快的速度对敌人发动斩击
自动标记范围内的敌人，被标记的范围内的敌人在最后将引发爆炸
记录敌人之前会先进入僵直状态， 按住技能键可控制施放时机
装备的武器不同， 附加的效果也不同
- [武器效果] -
巨剑 : 可追加蓄气， 并根据蓄气情况增加攻击力
短剑 : 可吸附敌人， 按技能Lv提高增加吸附力
太刀 : 增加1次多段攻击， 并附加出血效果
光剑 : 增加1次多段攻击， 并附加感电效果
钝器 : 附加眩晕效果







极·神剑术(瞬斩)[回到顶部]

使用神剑术专用的飞剑攻击前方的敌人
被飞剑击中的敌人受到剑气的影响，会受到第二次伤害







万剑归宗[回到顶部]

只有登顶剑神之境，才可施展的最高级剑术。在特定条件下，可随心所欲地操纵武器攻击敌人
- 以下状况可发动[穿云刺] -
普通攻击、 跳跃攻击、 前冲攻击
技能[三段斩]、 [猛龙断空斩]、 [流心 : 刺]、 [流心 : 跃]、 [流心 : 升]、 [银光落刃]、 [幻影剑舞]、 [极 · 神剑术 (破空斩)]、 [破军斩龙击]
万剑归宗进入最后阶段时， 根据领主、 精英、 HP最高的敌人的先后顺序抓取敌人， 发动乱舞攻击后爆炸







无形剑意[回到顶部]

剑意进一步强化， 增加基本攻击力和转职技能攻击力。 (装备不同类型的武器时， [流心 : 刺]、 [流心 : 跃]、 [流心 : 升]中2个存在特殊附加效果的技能攻击力增加)
同时， 利用无形剑， 装备不同类型的武器时分别对[流心 : 刺]、 [流心 : 跃]、 [流心 : 升]其中2个技能附加特殊效果
不同类型武器的特殊附加效果
[短剑]
[流心 : 刺] : 发动无形剑-短剑斩击 (追加下斩攻击时， 也可以发动无形剑斩击； 空中施放时发动连续斩击)
[流心 : 跃] : 发动无形剑-短剑斩击
[太刀]
[流心 : 跃] : 发动无形剑-太刀连续斩击
[流心 : 升] : 发动无形剑-太刀连续斩击
[钝器]
[流心 : 刺] : 产生无形剑-钝器下斩冲击波
[流心 : 跃] : 产生无形剑-钝器下斩冲击波
[巨剑]
[流心 : 刺] : 用无形剑-巨剑发动强化攻击 (追加下斩和空中施放时， 也可以用无形剑发动强化攻击； 追加下斩时生成旋风)
[流心 : 升] : 用无形剑-巨剑发动强化攻击
[光剑]
[流心 : 跃] : 用无形剑-光剑发动强化攻击
[流心 : 升] : 发动无形剑-光剑斩击







极·神剑术（无形斩）[回到顶部]

剑魂发动强力斩击， 同时乱舞无形剑
无形剑在一定范围内进行连续斩击， 然后爆炸并消失
被剑魂击中的敌人离开无形剑斩击区域时， 无形剑会对该敌人发动追击







万剑极诣·开天斩[回到顶部]

将无形剑变换为五种武器，发动连续攻击
使用钝器下劈，再用巨剑发动强力斩击。然后跳起使用短剑和太刀释放出剑气，落地后发动多次斩击，最后集中所有力量凝聚出光剑，发动终结攻击







光速之剑[回到顶部]

使用光剑系武器攻击敌人时，可以增加光剑的攻击速度






鬼门关[回到顶部]

在即将受到攻击前的一瞬间使用[格挡]，可以完全免疫伤害，并且反射敌人的远程攻击






武器节制[回到顶部]

掌握武器耐久度的极限点，可以减缓武器耐久度的下降速度，在[破极兵刃]已施放状态下亦有效
