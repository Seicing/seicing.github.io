长枪精通[回到顶部]

熟练掌握长枪技巧， 增加物理攻击力、 攻击速度、 命中率， 并减少技能冷却时间， 但[流云幻灭]与[极影无形杀]的冷却时间不会减少
学习此技能后， 可以在空中强制施放[落凤枪]





行云 : 沐[回到顶部]

增加基本攻击力和[刺击]、 [落凤枪]、 [连环枪]等转职技能的攻击力





无双枪术[回到顶部]

将斗气附加于枪上， 给予敌人这次伤害量一定比率的追加伤害
[无双枪术]的比率伤害会以同样的比率作用在装备的追加伤害效果上







行云诀[回到顶部]

使用后， 保留当前技能的幻影， 并用[后跳]脱身后可以自由移动
在不强制中断攻击技能 (静止状态、 普通攻击)的情况下， 按下技能键会发动[后跳]而不是[行云诀]
行云系列的技能可以用[行云诀]加上方向键的方式使用
可以用[行云诀]加方向键的方式施放的技能 :
[行云 : 风]
[行云 : 疾]
[行云 : 落]





行云 : 风[回到顶部]

行云系列的技能， 在行走、 前冲、 普通攻击、 施放其他攻击技能时， 都可以使用。 使用可以把敌人吸进去的旋风对敌人进行刺击
在施放其他攻击技能过程中使用， 可以留下正在攻击的幻影， 并施放[行云 : 风]
开始的第一击有推开靠近的敌人的效果
可以用[行云诀] + →施放[行云 : 风]
行走、 普通攻击、 前冲、 前冲攻击时可以使用







行云 : 疾[回到顶部]

行云系列的技能， 在行走、 前冲、 普通攻击、 施放其他攻击技能时， 都可以使用。 施放向下的横扫攻击
在施放其他攻击技能过程中使用， 可以留下正在攻击的幻影， 并施放[行云 : 疾]
可以以基础浮空力的一定比率使倒地的敌人再次浮空
可以用[行云诀] + ↓施放[行云 : 疾]
行走、 普通攻击、 前冲、 前冲攻击时可以使用







行云 : 落[回到顶部]

行云系列的技能， 在行走、 前冲、 普通攻击、 施放其他攻击技能时， 都可以使用。 使用后， 会跳至空中投掷长枪
在施放其他攻击技能过程中使用， 可以留下正在攻击的幻影， 并施放[行云 : 落]
投枪会使敌人处于超级控制状态， 并且浮空的敌人的控制时间更久
跳跃投枪时， 可以左右移动
可以用[行云诀] + ↑施放[行云 : 落]
行走、 普通攻击、 前冲、 前冲攻击时可以使用







双重刺击[回到顶部]

瞬间施放力量聚于一点的二连刺击。 枪尾刺击拥有更强的威力
近距离攻击和枪尾攻击不会同时命中一个敌人， 判定重叠时， 会优先判定为枪尾攻击







旋风枪[回到顶部]

向前方旋转长枪， 对敌人进行猛烈的打击。 按下前方向键时， 前进距离和旋转次数都会增加
在决斗场无法攻击倒地的敌人







无畏波动枪[回到顶部]

持枪直线突进进行刺击， 并将敌人刺穿在枪上， 随后敌人会被弹开， 并受到额外伤害
按下前方向键时， 会增加突进距离







螺旋波动枪[回到顶部]

向前方大幅挥枪激起气流，连续攻击敌人







升龙破空枪[回到顶部]

横扫前方、 挑飞敌人、 并大力刺向浮空的敌人







狂龙怒啸[回到顶部]

枪身举至头顶高速旋转， 把命中的敌人吸至中心
提升技能Lv可增加旋转速度







夺命雷霆枪[回到顶部]

用长枪连续刺击敌人要害。 敌人被枪尾刺击命中时， 会积累受创效果， 最后用横斩对积累受创效果的敌人造成额外伤害
最后横斩的攻击力与连续刺击的次数成正比







行云 : 启[回到顶部]

流畅地循环气力， 增加基本攻击和所有技能的攻击力。 而且特定技能可以增加更多的攻击力





流云幻灭[回到顶部]

横扫前方并聚集敌人后， 放出使用决战者的技能的分身， 投出聚集斗气的长枪， 虚弱大范围敌人
根据技能Lv的不同， 会产生下列附加效果
- Lv3 : 增加斗气长枪的攻击力
- Lv6 : 技能使用过程中处于无敌状态
- Lv9 : 斗气长枪的超级控制时间会受到[行云 : 落]控制时间的影响， 会增加一定比率的控制时间







风火燎原[回到顶部]

可以在地面或空中使用
在地面使用时， 向前方跳跃， 通过追加按键投出集束长枪
施放瞬间可以用前后方向键调整跳跃距离， 达到跳跃的最高点时， 按下技能键或X键可以发射集束长枪
施放[风火燎原]后， 可以接着使用[落凤枪]、 [流光无影闪]







三一斩月[回到顶部]

向前方大幅挥动枪身， 进行新月攻击。 被击中的敌人会聚集在新月的中间
近距离的敌人受到枪身的攻击后会被推开







行云 : 冥[回到顶部]

经过重重修炼， 通过魔枪驾驭行云的技术达到顶峰
学习后， 会增加特定技能的攻击力和物理暴击率
施放[行云诀]时， 会赋予无敌时间。 施放刺击系列技能时， 可以强制中断并连接施放其他刺击系列技能。 并且施放特定技能时， 将追加与原攻击力成一定比率的幻影攻击
- [可追加幻影攻击的技能] -
- [行云 : 风]、 [行云 : 疾]、 [行云 : 落]
- [刺击]、 [双重刺击]
- 地面普通攻击
- [可连接使用的刺击系列技能] -
- 可以以[刺击]、 [双重刺击]、 [无双突刺]的顺序连接使用
- 连接使用时， 技能的攻击速度会增加
- 学习技能后， [行云 : 风]的攻击速度会增加
- 学习技能后， 在[后跳]或[行云诀]状态下可以使用普通攻击





无双突刺[回到顶部]

施放2次牵制刺击后， 追加2次强力的终结刺击
第3~4击判定为枪尾攻击时， 会造成更大的伤害







流光无影闪[回到顶部]

高速突进， 聚集前方的怪物后， 施展强力的终结击
可以在空中施放







极影无形杀[回到顶部]

施放超长攻击距离的刺击控制敌人， 随后突进过去展开斩击与刺击的乱舞刺击
刺击后的突进碰到敌人时， 角色会即刻在原地进行乱舞
该技能无法用行云系技能强制中断
输入前方向键后， 终结一击会向前突进







一身是胆[回到顶部]

千魂·决战者的行动如同幻影，更加游刃有余的驾驭行云的技术，增加基本攻击力和转职技能攻击力，变更部分技能效果
[行云诀]
- 行云诀发动期间，可以使用其他攻击技能
[无畏波动枪]
- 施放[无畏波动枪]时，可以立即强制中断并连接行云系列技能；强制中断施放时，幻影分身会将敌人拉近








双龙流云灭[回到顶部]

注入斗气，用力向地面下劈，然后与仿佛有生命般的幻影共同攻击。可以在地面和空中施放







问鼎·千军破云[回到顶部]

将自身斗气聚于一处，贯穿万物
施放时，千魂·决战者大幅挥动魔枪释放斗气，消失在迷雾中。然后，幻影发动乱舞攻击，滞空的千魂·决战者聚集所有幻影制作的巨型魔枪，贯穿地上的一切







旋转冲击[回到顶部]

快速转动长枪造成伤害
