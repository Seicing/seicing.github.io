暗杀教义[回到顶部]

影舞者特有的暗杀术。 进行背后攻击时， 敌人的被击方向不会改变， 强制控制敌人时， 敌人的方向也不会改变。 若技能的首次攻击判定为从背后命中敌人， 则该技能施放结束为止都维持背击判定。 另外， 普通攻击、 [弧光闪]、 [天诛]、 [碎踝]的攻击效果会产生变化， 且增加暴击伤害
[弧光闪] : 移动速度不变， 通过追加操作可以进行一次追加攻击。 使用[弧光闪]进行背后攻击时， 敌人会转向攻击的方向
[天诛] : 变更最后一击的特效， 并且对无法抓取的敌人进行踢腿攻击
[碎踝] : 使被击中的敌人向后回头
影舞者基本攻击第三击 : 使被击中的敌人向后回头







暗杀之匕首精通[回到顶部]

使用匕首系列武器时， 可以增加武器物理攻击力和命中率。 攻击时对敌人造成的僵直时间提高至双剑水准
在决斗场中， 不会增加僵直时间






暗杀领域[回到顶部]

影舞者先贤为了弥补影舞者攻击范围不足而创造的领域技能。 指定的技能命中敌人背部时， 会召唤暗杀分身攻击附近的敌人
暗杀分身的攻击力等同于施放的技能攻击力。 再次按技能键时， 取消技能
随着技能Lv的增加， 背击时可以产生暗杀分身的技能的种类也会增加
Lv1 - 基本攻击第4击、 [天诛]、 [割喉]、 [夺魂]
Lv2 - [穿心]
Lv3 - [追命]
Lv4 - [刃舞]、 [潜影刺击]
Lv5 - [影戮]
Lv6 - [瞬杀]
Lv7 - [绝命飞刃]
Lv8 - [八荒影杀]
Lv9 - [死亡连舞]
Lv10 - [剜心]、 [幽冥链狱 : 索命]







影遁[回到顶部]

召唤出挑衅敌人的暗杀分身， 并且在一段时间内变为隐身状态进行突进
暗杀分身的HP与施放技能时的角色HP成正比
隐身状态下暴击率会大大增加。 进行攻击时， 会立即解除隐身状态。 学习[潜行暗杀]后， 可以在施放技能过程中使用[影遁]， 并且在隐身状态下攻击敌人必会获得背击命中的效果







暗杀信条[回到顶部]

暗杀者的信条， 增加暴击率和暴击伤害
暗杀者的五种美德
忍耐、 集中、 冷静、 沉着、 任务至上。 (节选自教本《暗杀的定式》)







追影步[回到顶部]

向前快速移动， 若移动途中碰到敌人， 将会移动到敌人的背后。 施放该技能时， 可以强制中断某些指定技能的施放。 按向后方向键可以回到原地。 某些技能被强制中断时， 会制造暗杀分身完成未完成的动作
可以强制中断的技能 - [弧光闪]、 [夺魂]、 [穿心]、 [刃舞]、 [潜影刺击]
强制中断技能时， 产生暗杀分身的技能 - [天诛]、 [影戮]、 [八荒影杀]、 [死亡连舞]







割喉[回到顶部]

使用武器向下攻击敌人。 如果命中敌人背部， 将会进行追加攻击。 此技能可在空中使用， 在空中使用时， 会垂直下落并向下攻击敌人。 技能命中敌人后， 使用[夺魂]可以与其他技能形成连招







夺魂[回到顶部]

从地面垂直对敌人发出上挑攻击。 击中倒地或者浮在低空中的敌人可以挑起对方。 被击中的敌人会进入无视硬直的僵直状态
在决斗场对倒地的敌人使用时， 不会挑起对方， 而是让敌人浮在空中







穿心[回到顶部]

以锐利的角度刺向敌人的要害。 技能命中敌人背部时， 会发动剜出敌人心脏的要害攻击， 同时让敌人进入僵直状态
被要害攻击命中的敌人将会减少硬直最大值, 并使敌人背面朝向影舞者







追命[回到顶部]

利用离心力将抓取到的敌人凶狠地插在地上， 同时引发冲击波。 可以在空中使用该技能。 如果技能命中敌人背部， 冲击波的攻击力和范围会增加
在地面使用技能时， 跳跃较长距离进行攻击。 通过方向键可以调整中/远距离以及Y轴的移动方向
在空中使用技能时， 会把敌人插在正前方的地面上。 [后跳]过程中也可以使用空中[追命]







潜影刺击[回到顶部]

快速挥斩两次后， 移动到敌人背后， 最后强力刺击敌人要害。 从背后命中敌人时， 可以造成更深的伤口







刃舞[回到顶部]

猛烈挥舞手中的剑刃， 对周围的敌人进行攻击。 施放过程中， 连续按下攻击键可以增加回旋攻击次数。 在回旋过程中按下Z键， 可以快速施放最后的斩击。 最后一击会把敌人聚集在一起， 可以通过方向键来调整最后一击的方向
回旋斩击命中敌人时， 会使敌人背面朝向影舞者







影戮[回到顶部]

通过华丽而致命的乱舞斩击毁灭敌人
命中敌人背部时， 会施放强力的斩击将敌人斩为两半
命中敌人时， 使用[追影步]可以让暗杀分身继续进行该技能的剩余动作
在决斗场中， 只有当影戮攻击命中目标时， 可以用追影步强行停止分身







瞬杀[回到顶部]

悄无声息地接近敌人进行暗杀。 靠近敌人背后进行暗杀时， 会施放致命一击把敌人切成4块。 被暗杀的对象的HP越少， 受到的伤害越高







暗杀之心[回到顶部]

死亡舞会所属要员必须掌握的暗杀术， 可以准确的从背后命中目标的要害
技能命中敌人背部时， 会增加攻击力和暴击率






影斩·乱舞[回到顶部]

梦魇会在大范围内进行快速移动并斩击敌人。 每次斩击都会制造一个暗杀分身进行攻击， 最后在暗杀分身和梦魇的夹击下颠覆空间。 连续按攻击键， 可以加快施放速度。 按前或下方向键， 可以在原地发动最终一击
技能达到Lv3及以上时， 使用技能可以在一段时间内获得Buff效果
Lv3 Buff效果 : [暗杀之匕首精通]技能Lv +3
Lv6 追加Buff效果 : 命中率/回避率增加
Lv9 追加Buff效果 : 技能冷却时间 -10%
[要我一个打一百个？ 好吧， 那我就放过观众， 反正也没有报酬。]







绝命飞刃[回到顶部]

投掷匕首， 将匕首掷入敌人的身体里， 可在空中进行投掷。 匕首会在敌人身体内滞留一段时间， 此时再次输入技能键， 影舞者会接近敌人， 随后拔出匕首进行一次攻击
投掷匕首的动作会让敌人的被击方向变为后方， 并将敌人击退到远处
技能命中敌人时， 会使敌人背朝影舞者







八荒影杀[回到顶部]

对敌人进行斩击， 使敌人进入僵直状态。 随后制造暗杀分身， 从八个方向同时斩击敌人。 如果技能命中敌人背部， 最后一击时会进行追加攻击。 施放过程中， 可以通过[追影步]制造分身， 并重现技能







潜行暗杀[回到顶部]

增加自身基本攻击力和技能攻击力。 背击攻击命中后， 会在一段时间内增加队友对命中敌人的暴击伤害
每次背击攻击命中时， 重新计算效果持续时间
技能施放过程中可以使用[影遁]， [影遁]的隐身状态下， 进行的攻击会被判定为背击攻击
技能施放过程中使用[影遁]时， 扩散死亡烟雾， 烟雾碰到敌人时也会施加[潜行暗杀]效果






死亡连舞[回到顶部]

抓住敌人后， 使用迅速、 干净利落的动作对要害进行连续攻击。 从背后进行攻击时， 会施放强力的追加攻击。 技能命中时使用[追影步]， 暗杀分身会完成未完成的动作







剜心[回到顶部]

用绚丽旋转的步伐对前方敌人进行攻击。 如果命中敌人的正面， 用强烈的旋转力施加重击。 如果命中敌人的背部， 则会将敌人的要害握在手里， 随后引起爆炸







幽冥链狱 : 索命[回到顶部]

使用末端连接锁链的匕首， 对敌人进行致命的攻击
技能的过程中按下技能键时， 可以射出锁链。 可以在空中射出锁链， 也可以强制中断特定技能后射出锁链。 如果锁链插中敌人， 可以通过追加操作进行其他的动作
未按键 : 把敌人拉到自己面前
X键 : 靠近敌人并进行攻击
在敌人背后按下X键 + 方向键 : 靠近敌人后转换位置， 并进行强力一击
Space键 : 把锁链插到地面上， 束缚敌人一段时间。 此动作有另外的冷却时间
再次按下技能键 : 施放强力终结攻击
可以强制中断的技能 : [翔击]、 [切割]、 [弧光闪]、 [影袭]、 [碎踝]、 [追影步]、 [夺魂]、 [穿心]、 [刃舞]、 [潜影刺击]、 [影斩·乱舞]
可以强制中断特定技能的后僵直并发射锁链的技能:
[绝命飞刃] (最后二击)、 [剜心] (要害袭击动作)、 [瞬杀] (转刀动作)







暗影禁忌[回到顶部]

隐夜·影舞者利用他人的影子施展禁忌暗杀术，在任何人都无法察觉的情况下，除掉敌人
增加隐夜·影舞者的基本攻击力与技能攻击力，使部分技能产生附加效果
暗杀领域
- 增加暗影领域范围
- 背击成功时若存在多个敌人， 则增加每次召唤的暗影分身的数量
穿心
- 影舞者的所有转职技能的终结动作时可以预先按该技能键
- 预先按键后发动技能的状态下， 技能完全结束或施放[追影步]时， 出现在该技能命中的敌人中最强的敌人背后， 穿刺敌人的心脏







影缚追魂锁[回到顶部]

投掷暗影锁链，抽取敌人影子，对其行追魂攻击
暗影锁链命中敌人背后时，抽取影子后快速移动，切割影子并使其爆炸。影子爆炸瞬间，所有伤害都会瞬间转移到敌人本体，造成致命伤害
背后攻击时，若敌人影子贴近隐夜·影舞者，则后空翻拉开距离后再开始攻击







无间影狱·噬灭[回到顶部]

隐夜·影舞者觉醒自身暗影领域， 融入蔓延到周围的暗影地带， 隐藏身影
用锁链环绕周围一带后， 用暗影匕首斩击敌人。 最后， 挥动暗影匕首施展3次致命攻击， 最后投掷匕首进行暗杀
暗影地带内的敌人都是隐夜·影舞者的暗杀对象，所有攻击均适用背后攻击
施放技能时，无法使用[影遁]，无法强制中断并施放影舞者转职系列技能
选择[幽冥链狱 : 索命]使用三次觉醒技能时， 可以使用三次觉醒技能代替[幽冥链狱 : 索命]的终结攻击
三次觉醒技能在冷却中、[幽冥链狱 : 索命]终结攻击己发生或[幽冥链狱 : 索命]未激活的状态下，无法使用三次觉醒技能
