抓轰炮[回到顶部]

基本攻击、 前冲攻击、 跳跃攻击变更为物理独立攻击力； 对无法抓取或固定型敌人进行抓取时， 可以强力推开敌人并给予敌人一定伤害
- [发动抓轰炮效果的技能] -
背摔 : 强力推开敌人并攻击
抛投 : 与敌人力量制衡并在一定时间后推开 (控制时间与基本的抛投相同)
空绞锤 : 对敌人发动强力的踢击并跳跃
霹雳肘击 : 用肘部进行强力的攻击
螺旋彗星落 : 旋转中强制控制敌人后推开敌人
地狱摇篮 : 肩部攻击， 并在旋转攻击后进行强力的踢击并跳跃
空绞连锤 : 原地旋转攻击后， 进行强力的踢击并跳跃
死亡摇篮 : 肩部攻击， 并在旋转攻击后进行强力的踢击并跳跃
末日摇篮 : 旋转一圈攻击敌人并将其强力推开
推开敌人时会生成冲击波， 冲击波的攻击力与技能本身的攻击力相同， 但施放[雷霆背摔]、 [摔技强化]时， 不会出现[抓轰炮]效果







摔技强化[回到顶部]

使用[金刚碎]、 [背摔]、 [抛投]、 [霹雳肘击]、 [空绞锤]、 [螺旋彗星落]、 [地狱摇篮]、 [裂石破天]、 [末日风暴]、 [死亡摇篮]、 [空绞连锤]技能时， 可以增加冲击波、 间接攻击的攻击范围和攻击力






暴力抓取[回到顶部]

使[背摔]、 [抛投]、 [霹雳肘击]、 [空绞锤]、 [螺旋彗星落]、 [地狱摇篮]、 [裂石破天]、 [末日风暴]、[空绞连锤]、[死亡摇篮] [末日摇篮]、 [风暴之舞]、 [苍宇彗星落]的技能抓取动作出现霸体判定， 并且抓取命中率变成100%， 同时还增加基本攻击和抓取技能的攻击力， 效果持续一定时间







滑行抓取[回到顶部]

在前冲状态下使用[背摔]、 [抛投]、 [螺旋彗星落]、 [空绞锤]、 [地狱摇篮]、 [末日风暴]、 [裂石破天]、 [死亡摇篮]、 [苍宇彗星落]技能时， 可以滑行抓取敌人
滑行抓取敌人时， 可以用方向键进行上下滑行； 技能Lv越高， 滑行速度越快







强力抱摔[回到顶部]

强化柔道家的基本攻击和抓取技能， 增加攻击力和对敌人造成致命一击的概率
攻击力增加的技能 : [背摔]、 [抛投]、 [霹雳肘击]、 [空绞锤]、 [螺旋彗星落]、 [地狱摇篮]、 [裂石破天]、 [末日风暴]、 [空绞连锤]、 [死亡摇篮]、 [末日摇篮]、 [风暴之舞]、 [苍宇彗星落]






格斗专精[回到顶部]

学习技能后， [折颈]、 [霹雳冲击]、 [野蛮冲撞]可以获得[强力抱摔]、 [暴力抓取]、 [抓取奥义]、 [极手奥义]的攻击力增加效果。 [折颈]、 [霹雳冲击]、 [野蛮冲撞]攻击时， 使敌人受到[抓取奥义]的所受伤害增加效果
以下技能连续抓取敌人时变得更加有利
[背摔] : 攻击中可以强制中断并施放[霹雳肘击]
[霹雳肘击] : 减少把敌人击飞的力量和攻击后的延迟
[空绞锤] : 被向下捶击击倒的敌人更久地躺在地上
[螺旋彗星落] : 向下捶击敌人后， 可以强制中断并施放[空绞锤]和[空绞连锤]
[空绞连锤] : 挑飞向下捶击的敌人， 使之浮空






无情抓取[回到顶部]

[抓轰炮]施放过程中可以使用的技能。 攻击成功时， 对敌人造成较大程度的僵直状态， 并且可以连接使用[折颈]或其他抓取类技能
在地面或空中施放时， 拥有不同的攻击姿态
- 在地面施放时 : 暴力拦截 -
- 在空中施放时 : 抓取空翻、 暴力关节 -
暴力拦截 : 使用肩膀大力冲撞敌人
抓取空翻 : 抓取敌人后， 施展强力的空翻踢
暴力关节 : 空翻后， 在一段时间内追加按下技能键时， 快速地从空中向下拳击
拥有特有的重复等待时间， 并且拥有最大可使用次数
消耗最大可使用次数后， 一段时间后会自动恢复







抛投[回到顶部]

抓取敌人并向前方抛摔， 使其对撞到的敌人造成一定伤害； 被抛摔的敌人只有在撞到墙壁或地面后才会受到伤害
抓取敌人后， 再次按攻击键就可以抛摔敌人， 此时可以通过方向键指定抛摔方向
敌人撞到墙壁和撞到地面受到的伤害不同， 撞到墙壁落地时不会出现第2次伤害
抓取判定 : 可以抓取霸体和防御状态的敌人， 但对少数敌人无效
在决斗场中， 技能Lv越高， 抓取持续时间越长







野蛮冲撞[回到顶部]

向前滑行的同时用肩部猛力撞击敌人
命中敌人后， 可以立即连接[背摔]、 [抛投]、 [霹雳肘击]、 [螺旋彗星落]、 [地狱摇篮]、 [末日风暴]、 [死亡摇篮]、 [末日摇篮]、 [霹雳冲击]、 [风暴之舞]、 [苍宇彗星落]
[野蛮冲撞]蓄气后可以向敌人发出更快更威力的撞击， 达最大蓄气时有霸体判定
若前冲时使用， 则可以边向前滑行边蓄气







折颈[回到顶部]

可以强制被自身击中的敌人转身， 并使转身后的敌人进入短暂的僵直状态； 转职为柔道家时， 增加僵直时间
无法使固定型敌人转身
在决斗场中， 命中时使敌人浮空并进入僵直状态







霹雳肘击[回到顶部]

抓起倒地的敌人， 并用强劲的肘击攻击该敌人使其受到巨大伤害， 同时还会生成冲击波使周围的敌人受到一定的伤害
技能结束后， 自身会进入较长时间的僵直状态







空绞锤[回到顶部]

在空中用双腿夹住敌人并砸向地面。 攻击无法抓取的敌人时， 踢中敌人后向上跳起







霹雳冲击[回到顶部]

用肘部攻击前方敌人， 然后旋转跳起， 用膝盖撞击敌人
攻击成功时， 产生强力冲击波， 对周围敌人造成伤害。 (直接命中的敌人不受冲击波伤害)
肘部攻击和膝盖撞击前， 按向前方向键， 可以增加前进距离
肘部攻击成功时， 角色停止移动； 膝盖撞击成功时， 可以强制中断当前动作并立即连接部分技能
- [可连接技能] -
[旋风腿]、 [空绞锤]、 [裂石破天]、 [空绞连锤]、 [死亡摇篮] (随机应变)、 [苍宇彗星落] (随机应变)







螺旋彗星落[回到顶部]

旋转一圈， 并抓取一定范围内的所有敌人向空中跳跃， 然后再将敌人用力砸向地面使其受到极大伤害， 跳跃时可以用方向键改变砸击方向
敌人间发生碰撞时， 会相互给予对方一定伤害； 砸到地面时， 会对周围敌人造成一定伤害
技能发动时， 按技能键的长短间隔会影响跳跃高度， 而跳跃高度会影响跳跃移动的距离
抓取判定 : 可以抓取霸体和防御状态的敌人， 但对少数敌人无效







地狱摇篮[回到顶部]

抓取敌人后， 先旋转着对敌人发动连续的背摔， 然后再向敌人踢出强力一脚并使自身向上跳跃
发动最后一击时， 可以强制中断并立即施放[旋风腿]、 [空绞锤]、 [裂石破天]、 [空绞连锤]
发动连续背摔时， 可以用方向键控制移动方向
抓取判定 : 可以抓取霸体和防御状态的敌人， 但对少数敌人无效







裂石破天[回到顶部]

快速从空中下坠并抓取一名敌人， 可以使该敌人和周围的敌人随自身向前拖行一段距离并使敌人受到多段攻击伤害
下坠时会生成冲击波并给予敌人一定伤害， 最后巨岩会发生爆炸并给予敌人巨大伤害。 施放者落地后会处于霸体状态， 直到技能结束
攻击固定型敌人时， 无法向前拖行， 但可以强制控制攻击范围内的固定型敌人。 爆炸结束后， 解除强制控制并造成大量伤害
落地前， 输入向后方向键， 可以大幅减少下降角度和抓取敌人后的移动距离
抓取判定 : 可以抓取霸体和防御状态的敌人， 但对少数敌人无效
- [随机应变] -
攻击判定 : 无抓取判定， 下坠并强力攻击敌人。 输入后方向键， 可以减少下降角度







抓取奥义[回到顶部]

将抓取动作提升到极致， 并增加暴风眼的基本攻击和抓取类技能的攻击力、 物理暴击率、 命中率和抓取时的回避率
被抓取的怪物所受伤害增加






末日风暴[回到顶部]

抓取一名敌人后快速旋转并在自身周围生成一个巨大风暴， 风暴可以吸附碰到风暴的敌人并使敌人受到持续伤害。 施放者最后一击可以跳至空中并用强力一拳砸向地面， 可以产生冲击波攻击周围的敌人
可以用方向键控制风暴的移动方向， 除固定型敌人外， 风暴可以吸附其他任何类型的敌人； 对无法抓取型敌人施放[末日风暴]时， 可以生成无法移动的暴风
旋转过程中按技能键或者z键， 可以立即施放最后一击
Lv3 : [末日风暴]抓取动作期间进入无敌状态
Lv6 : [末日风暴]旋转移动速度增加30%
Lv9 : [末日风暴]最后一击攻击力增加10%







空绞连锤[回到顶部]

在空中用双腿夹住敌人原地旋转， 使被抓取的敌人及其周围的敌人受到旋转物理伤害
旋转后抓取敌人再用力砸向地面， 使敌人受到伤害并浮空
向上跃起继续夹住浮空的敌人并再次用力将其砸向地面， 会使敌人受到追加伤害
无法抓取敌人时， 在旋转攻击后会将敌人踢飞， 并使其受到伤害







死亡摇篮[回到顶部]

抓取敌人后， 腾空跳起并将敌人砸向地面， 然后向前翻滚的同时把敌人拉到一起， 最后再向敌人踢出强威力的一脚
发动最后一击时， 可以强制中断并立即施放[旋风腿]、 [空绞锤]、 [裂石破天]、 [空绞连锤]
旋转时可以用方向键控制前后移动方向， 按下方向键可以在原地旋转并发动最后一击
抓取判定 : 可以抓取霸体和防御状态的敌人， 但对少数敌人无效
- [随机应变] -
攻击判定 : 旋转并腾空跳起， 攻击敌人。 按下方向键可以在原地发动攻击力







极手奥义[回到顶部]

抓取成功时， 可增加基本攻击力和抓取技能的攻击力， 该Buff效果持续一定时间
- [适用技能] -
[背摔]、 [抛投]、 [霹雳肘击]、 [空绞锤]、 [螺旋彗星落]、 [地狱摇篮]、 [裂石破天]、 [末日风暴]、 [死亡摇篮]、 [空绞连锤]、 [末日摇篮]、 [风暴之舞]、 [苍宇彗星落]






随机应变[回到顶部]

精通抓取技的风暴女皇可以根据敌人的状态控制自己的技能
学习该技能后， 部分抓取技能变更为攻击技能
即使变更为攻击技能， 转职和觉醒被动效果依然生效
变更的技能 : [裂石破天]、 [死亡摇篮]、 [末日摇篮]、 [苍宇彗星落]






末日摇篮[回到顶部]

向前旋转， 并且在抓取敌人后继续疾速旋转对敌人造成伤害
扔出敌人之前若按向下方向键， 则原地扔出敌人； 若按向前方向键， 则能将敌人扔得更远
攻击不可抓取的敌人时， 第一次攻击时会强制控制敌人， 旋转一圈后施放[抓轰炮]
- [随机应变] -
攻击判定 : 旋转一定次数同时进行攻击， 然后发动最后一击







风暴之舞[回到顶部]

围绕敌人高速旋转并攻击其关节， 在一段时间后对敌人造成大量伤害







苍宇彗星落[回到顶部]

抓取敌人并使用暴力上勾拳将其打向高空， 然后高高跃起， 抱住敌人一起疾速旋转后将其用力砸向地面使其受到极大伤害
上勾拳以后连续按下技能键， 可以更快地把敌人砸向地面
抓取判定 : 可以抓取霸体和防御状态的敌人， 但对少数敌人无效； 若未抓取则直接发动最后一击
- [随机应变] -
攻击判定 : 猛踢敌人， 然后跳起强力下砸攻击敌人。 在空中也可以施放







光芒之翼[回到顶部]

战斗舞台的女皇-归元·柔道家与他的弟子莱茜&露西姐妹组成的终极组合
她们为观众展现更加华丽和超凡的战斗
增加除[背摔]以外的归元·柔道家基本攻击力和转职技能攻击力，部分技能附加额外效果
背摔
- 可抓取敌人：莱茜向倒地的向敌人使出月面宙反，造成额外伤害后弹回
- 无法抓取敌人：露西向命中的敌人使出飞身踢， 造成额外伤害后延长控制时间
野蛮冲撞
- 不论是否蓄气，抓取时均发动霸体护甲，在空中时也可以使用该技能
- 空中施放时，沿斜线落下，向敌人踢击，攻击后可以立即中断并使用空中抓取技能
- 准备动作时按方向键可以调整落下角度




送葬舞步[回到顶部]

光芒之翼莱茜&露西组合的招牌动作
柔道家将敌人过肩摔后，莱茜和露西交替攻击，对敌人发动终结攻击
根据抓取敌人的状态变更技能效果
- 可抓取：柔道家将敌人过肩摔后，莱茜突进后用金臂勾将敌人击飞到空中，然后露西快速抓取浮空敌人后使用翻滚打桩摔
每次攻击会产生冲击波，对周围敌人造成相同伤害
过肩摔时，可以按上方键将莱茜的金臂勾变更为飞身踢，远远踢飞敌人
- 抓轰炮：柔道家旋转攻击敌人，莱茜突进后使出金臂勾，然后露西对敌人使出飞身压
跳跃过程中也可以使用




女皇时代 · 辉煌舞台[回到顶部]

身在战斗舞台上的女皇使出最后的终结技
施放时，向地面发动致命捶击，产生冲击波，强制控制命中敌人，然后隐藏踪迹
随后莱茜&露西向最强的敌人发动双人飞身踢，U.F.E特设舞台出现
然后，幕后女皇登场，发动终极金臂勾，对敌人造成巨大伤害
终结攻击时，敌人的控制状态解除
跳跃过程中也可以施放该技能




背摔强化[回到顶部]

从敌人背后施放[背摔]时， 可以增加攻击力和暴击率
