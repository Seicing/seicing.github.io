唤魔：蛇腹剑[回到顶部]

进入地下城后， 武器将变成蛇腹剑形态， 契魔者可以使用其专属技能
基本攻击和[十字剑]等转职后技能附加出血功能， 并增加攻击力







唤魔：狂暴[回到顶部]

让魔王塔莫斯的精神支配着契魔者进行战斗
进入[唤魔 : 狂暴]状态后， 增加角色的基本攻击力和技能攻击力、 攻击速度和移动速度







魔剑控制[回到顶部]

获得控制魔剑普诺的力量， 可以灵活自如地操控魔剑
掌握该技能后， 增加武器攻击力和物理暴击率






蛇腹剑：破[回到顶部]

将蛇腹剑刺向地面， 使其向前延展并破地而出， 可对敌人造成多段攻击伤害







蛇腹剑：舞[回到顶部]

向前方挥舞3次蛇腹剑， 可对敌人造成伤害
随着挥舞次数的增加， 剑身会越变越长
第一击和第二击会把敌人推开， 第三击会缠住敌人的脚部后击倒并拉向角色身边







蛇腹剑：缠[回到顶部]

用蛇腹剑将前方的敌人缠绕并鞭笞
被鞭笞的敌人会受到多段攻击伤害并浮空
对于无法抓取的敌人只造成缠绕伤害







蛇腹剑：刺[回到顶部]

将蛇腹剑化为长鞭形态， 向前猛力一刺； 剑身延伸到一定距离后会收回
被刺中的敌人， 在收剑时会被拉回到角色身边







蛇腹剑：灭[回到顶部]

将蛇腹剑化为长鞭形态， 然后前后极速舞动对敌人造成伤害
连续按攻击键可以更快地舞动
按跳跃键可以中断技能







血饮狂舞[回到顶部]

将蛇腹剑化为长鞭形态， 并在前方进行大范围的疯狂舞动， 可造成多段伤害
连续按攻击键可以增加施放速度和攻击次数







碎魔剑[回到顶部]

剑身分裂成刀刃攻击敌人。 分裂的刀刃沿着一定轨迹移动后重新变回剑
在决斗场中， 减少攻击范围， 吸附攻击后的霸体不生效， 挥剑攻击无法攻击倒地的敌人







群魔乱舞[回到顶部]

召唤出多个自身幻影， 并令其攻击敌人
幻影会自动寻找敌人进行攻击； 当敌人数量少于幻影数量时， 每个敌人最多能承受5个幻影攻击
幻影会使用[蛇腹剑 : 破]、 [蛇腹剑 : 舞]、 [蛇腹剑 : 灭]、 [蛇腹剑 : 刺]、 [血饮狂舞]等技能







唤魔: 塔莫斯之袭[回到顶部]

挥剑斩击敌人， 随后命中者身边会出现魔王塔莫斯
在跳跃状态下也可以施放技能， 按↑ + 技能键， 可以发动强袭
强袭时， 快速向前飞一定距离， 在移动途中按技能键可以挥剑
出现的塔莫斯会对敌人进行疯狂的连续斩击， 最后以强力的交叉攻击结束并消失
若在强袭中没有击中目标， 则契魔者直接在落地点召唤出塔莫斯进行攻击







贪欲燔祭[回到顶部]

利用契魔者的转职技能击杀敌人时， 有一定几率产生云雾。 魔王塔莫斯会感知并吸收云雾
敌人的云雾将持续一定时间， 吸收云雾后， 可以获得获得增加基本攻击力和技能攻击力的Buff效果
该Buff有重叠次数限制






空绝斩: 千刃[回到顶部]

从异次元裂缝召唤出大量魔人克库斯进行攻击； 同时角色会逐渐化成恶魔形态冲向敌人， 最后将所有力量集中到蛇腹剑上进行强力的下斩攻击， 并引发爆炸
该技能达到Lv3、 Lv6、 Lv9时， 效果会有所增强
Lv3 : 攻击力增加10%
Lv6 : 克库斯攻击时， 会引发爆炸
Lv9 : 最终爆炸范围变为全屏







唤魔：逆天之普诺[回到顶部]

魔人普诺觉察出剑魔的弱点企图将其吞噬， 此时塔莫斯现身拯救剑魔， 并引发强烈的爆炸
施放后， 会释放出大量的克库斯魔人， 使周围敌人受到伤害
当普诺完全吞噬掉剑魔后， 塔莫斯会现身攻击普诺， 此时飞散的普诺碎片会对周围的敌人造成伤害； 当普诺最终被消灭时， 会使周围敌人受到大范围的爆炸伤害







汲血魔剑[回到顶部]

召唤塔莫斯， 塔莫斯和契魔者一起攻击敌人
攻击敌人时， 发动十字斩击







唤魔：弑神剑[回到顶部]

将弑神的力量赋予到蛇腹剑上， 强化技能效果
[蛇腹剑 : 破] : 增加攻击范围和攻击力， 更快射出刀刃
[蛇腹剑 : 舞] : 增加攻击力和柔软性， 可以强制中断当前施放中的蛇腹剑系列技能， 并立即施放[蛇腹剑 : 舞]或相反， 但是不适用于[蛇腹剑 : 缠]、 [弑神剑 : 极]、 [汲血魔剑]、 [空绝斩 : 地裂]
[蛇腹剑 : 刺] : 增加攻击范围和攻击力
[蛇腹剑 : 灭]、 [血饮狂舞] : 生成魔人的剑气， 造成更多的伤害
[蛇腹剑 : 缠] : 魔人灵体将会缠住敌人， 并造成持续伤害后爆炸
[唤魔 : 塔莫斯之袭] : 增加塔莫斯的攻击次数和最终一击的攻击力
基本攻击、 [十字剑]、 [蛇腹剑 : 破]、 [碎魔剑]、 [群魔乱舞]、 [唤魔 : 逆天之普诺]、 [汲血魔剑]、 [空绝斩 : 千刃]、 [空绝斩 : 地裂]、 [弑神剑 : 极]、 [弑神剑 : 诸神献祭]、 [蛇舞血轮剑]、 [弑神剑 : 恶魔孤岛] : 增加攻击力






空绝斩：地裂[回到顶部]

跳跃后， 将普诺变形为巨剑向下猛击形成冲击波。 被冲击波命中的敌人， 会被击倒在冲击波中心， 利用方向键可以控制跳跃的距离
方向键(无操作) : 原地
方向键(向前) : 中距离
方向键(向上) : 长距离







弑神剑：极[回到顶部]

魔人克库斯化身为镰刀， 抓住敌人左右甩动。 抓住敌人前， 可以利用方向键指定抛出敌人的方向







弑神剑：诸神献祭[回到顶部]

跃向空中， 与塔莫斯合体。 利用合体后的强大力量， 将普诺变形为最佳攻击形态向地面进行乱舞攻击， 之后再将普诺变形为巨剑形态刺向地面引发冲击波并使敌人进入僵直状态， 最后使用巨大的剑刃进行最终一击







魔源觉醒[回到顶部]

吸收魔人真正的力量， 增加基本攻击力和除转职技能攻击力 ([蛇腹剑 : 缠]除外)， 部分蛇腹剑技能赋予特殊效果
特殊效果只有学习[唤魔 : 弑神剑]后才会适用
[蛇腹剑 : 破] : 召唤的剑增加为6个
[蛇腹剑 : 缠] : 抓取并鞭笞敌人后，额外进行下劈攻击







蛇舞血轮剑[回到顶部]

将与塔莫斯合体的蛇腹剑插入地面， 击穿前方所有的敌人







弑神剑: 恶魔孤岛[回到顶部]

穿戴魔人的盔甲后， 与变身为剑的塔莫斯一起挥动蛇腹剑， 创造巨大的监狱
使用巨剑凝聚所有魔人的力量后， 摧毁整个监狱







蛇腹剑：狂[回到顶部]

增加[蛇腹剑 : 破]、 [蛇腹剑 : 缠]、 [蛇腹剑 : 舞]、 [蛇腹剑 : 刺]、 [蛇腹剑 : 灭]和[血饮狂舞]的攻击力， 效果持续一定时间






唤魔：封之印[回到顶部]

以牺牲自身HP为代价， 召唤魔人迦罗亚， 并射向前方敌人
命中后会将敌人黏住， 降低其移动速度， 同时生成迦罗亚的印痕在目标周围散落； 凡进入印痕范围内的敌人， 都会受到伤害， 并且降低移动速度
当印痕散落完毕时， 会引发爆炸并给周围敌人造成伤害







贪食者迦罗亚[回到顶部]

召唤贪食者伽罗亚攻击敌人。贪食者会自动搜寻猎物，若成功捕食敌人则会增加身躯大小
捕食次数或召唤时间达到上限时，会引发爆炸，对周围敌人造成伤害
每攻击一次敌人，则恢复剑魔一定量的HP







伤口恶化[回到顶部]

利用蛇腹剑系列技能攻击敌人时，可以使敌人进入伤口恶化状态；利用[唤魔：封之印]、[群魔乱舞]、[唤魔：塔莫斯之袭]、[贪食者迦罗亚]、[唤魔：逆天之普诺]、[空绝斩千刃]攻击该状态下的敌人时，可增加攻击力
