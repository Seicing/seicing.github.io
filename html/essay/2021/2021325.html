在战斗开始时，弓兵的作用是执行先制攻击，一排排的射手排成战列之后同时开弓向敌军射击，射出的箭矢将会覆盖一小片区域，主要是以敌军的方阵为目标，在射击后双方都会出现减员，这使得方阵开始缺少足以维持正常实力的兵力而出现弱点，甚至士气也会因此受到打击，在接下来的白刃战中溃不成军，匈奴弓骑兵使用腾格里的弓形态，她们的战术是利用自身的优势一波接一波地使用齐射骚扰敌军，当敌军开始进军时，她们会急速撤退，在撤退途中继续射击，敌人在拉锯战中疲于奔命的时候，埋伏的枪骑兵们会突袭因为疲劳而失去抵抗能力的猎物

特性 飞行状态(无视重力、无视击飞、无视跨越型地形障碍)
需要与其进行缠斗的精英级部队
行为 连续冲锋 冷却时间：3秒
短暂停止后快速向着一个方向突进，对一直线的敌人造成伤害，然后马上调转马头朝着反方向继续一次冲锋
每支冲锋造成120穿刺伤害
最熟练、最致命的匈奴战士被称为鞑坎(Tarkan)，这是匈奴人的武士阶级，他们不需要缴纳任何的税金，拥有自己的仆从和房产，也就是所谓的“战斗贵族”，他们在战斗时通常作为小队指挥官作战，作战命令被确实地传达到军队后，这群战士会身先士卒冲在最前线，以难以置信的战斗狂热撕裂敌人，为了使敌人更加恐惧，在战斗时更加可怖，他们会带上可怕的面具，面具内部有减缓痛楚的麻药，除非把他们的脑袋砍下来，这群战斗机器不会轻易地停止杀戮

散兵(Skirmisher)
等级类别 精锐 生命值 1145 掉落物品 无
斩击抗性 0 打击抗性 0 贯穿抗性 0 火抗 0 水抗 0 风抗 0 地抗 10 光抗 0 暗抗 0
特性 普通的步行敌人
拥有断末魔的敌人
行为 剑舞 冷却时间：3秒
进入准备攻击姿势后快速挥刀攻击，随后利用攻击的惯性大幅度翻转自身，利用螺旋作用力使出强力的劈斩攻击
挥刀造成70火属性伤害
劈斩造成250火属性伤害
烧夷 【断末魔】：散兵死亡后，手里的阿拉兹权现会掉落，让地面燃烧起来，让玩家进入灼烧状态
阿拉兹权现掉落到水里不会燃烧起来
和组成阵列的枪骑兵不同，散兵们利用自身单体作战能力优势独自作战，他们在匈奴军队担当侦察和骚扰，利用自身的速度优势突袭敌方的阵地，在冲锋的时候他们在侧翼包抄敌人，敌人四散而逃之后他们负责追击敌人，以及焚毁敌人的房屋掠夺战利品，他们为了更加敏捷地行动，一般不会使用其他士兵沉重的腾格里权现，而是使用阿拉兹权现，一种类似于火把的短剑，这使得他们可以轻易地挥动使用技巧杀伤敌人，并且焚毁房屋
