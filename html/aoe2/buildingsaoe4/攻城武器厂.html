更新日志[回到顶部]

发售版本
  • : v8324补丁后，火工技术研究场所从攻城武器厂移动到了大学
  • : v11009补丁后，精密枪械加成从25%下调到20%
第1赛季
  • 第一赛季补丁后，润滑轮轴移动速度加成从20%下调到15%
  • 第一赛季补丁后，几何学从大学移动到攻城武器厂，攻城武器从攻城武器厂移动到大学(蒙古除外)
  • 第一赛季补丁后，几何学资源成本从300木700金下调到100木225金，研究时间从90秒下调到45秒
  • : 第一赛季补丁后，攻城武器资源成本从150木350金上调到300木700金，研究时间从60秒上调到90秒
第2赛季
  • : 第二赛季补丁后，装填训练科技效果攻击速度加成从25%上调到33%
  • : 第二赛季补丁后，帝王时代的科技研发时间从15倍下调到了12倍
  • : 第二赛季补丁后，攻城武器的开发场地从攻城武器厂移动到了铁匠铺，获得了几何学
第3赛季
  • : 第三赛季补丁后，装填训练研究时间从90秒下调到60秒，总成本从1000下调到500
第4赛季
  • 第四赛季补丁后，攻城武器厂可以生产轻型冲车了
  • 第四赛季补丁后，获得轻质横梁科技，但几何学科技不再影响冲车
  • 第四赛季补丁后，手推炮、加农炮、风琴炮和重型火炮的木材成本都得到了下调
  • : 第四赛季补丁后，由于ui重制，现在普通的攻城武器厂也可以生产风琴炮了，而不是仅能在炮兵学院生产
  • 6.0.878补丁后，几何学伤害增加量从30%下调到了20%
第5赛季
  • : 第五赛季补丁后，可重复利用炮管科技改名为额外炮管，不再减少蜂巢炮的成本，而是增加2次多段攻击次数
  • : 第五赛季5822测试追加补丁后，额外炮管增加的2次多段攻击次数下调到1次，成本从150木350金上调到200木500金
  • 第五赛季5822测试追加补丁后，轻质横梁成本从100木225金上调到300木400金，研发时间从45秒上调到60秒，蒙古和德里苏丹国相关对应事项也随之改动
  • 8-2-218补丁后，轻质横梁科技攻击速度提升率从40%下调到了20%，轻型冲车在攻城武器厂生产的时间从30秒上调到了40秒
