The Battle of Ayn Jalut
剧本信息
发行版本 苏丹的崛起
文明 马穆鲁克苏丹国
苏丹国面临新的威胁：蒙古人来袭



开场故事

一股新的力量在昔日奴隶的旗帜下崛起。
马穆鲁克王朝诞生了。
尽管很骇人，法兰克人的蹂躏依然无法与蒙古人以雷霆之势给我们突然带来的恐怖相提并论。
就像不可阻挡的瘟疫一样，向我们扑来。
我们一个接一个地倒下。
巴格达...阿勒颇...大马士革。
我们的土地硝烟弥漫，血流成河。
但苏丹忽都斯不会袖手旁观，也不会在恐惧中逃跑，愿上帝怜悯他。
他召集了他的马穆鲁克军队，并行军前进。



剧本指南

依仗拜巴尔斯的援助，苏丹忽都斯发兵阻截可怕的蒙古人。

开始信息
  • 时代 : 城堡时代
  • 资源 : 无
  • 人口上限 : 200
  • 起始单位 : 拜巴尔斯、12突厥骑马弓兵

主要目的
  • 保证己方骑马弓兵存活
  • 在援军抵达之前削弱蒙古先锋部队
  • 阻止蒙古入侵
    • 击败敌方波数(0/5)
  • 阿富拉人民必须存活下来
    • 剩余村民(55)
游戏
  • 你在此项任务中无法获得任何村民或经济。
  • 带领拜巴尔斯和他的骑马弓兵开始游戏，在等待援军期间尽可能多地消灭蒙古部队。
  • 阻止蒙古入侵部队穿越山谷。阿富拉人民必须存活下来。
提示
  • 在与敌方单位交战时，使用最能克制敌人的单位类型来对付他们吧。弩手可以射穿重型单位；长矛手可以击溃骑兵。



玩家

玩家
  • 马穆鲁克苏丹国(Mamluk Sultanate) : 玩家必须从5波蒙古人的进攻中保护阿富拉，并击败所有的敌人赢得战斗，玩家不能发展任何经济，只能通过每波的援军来补充兵力

同盟
  • 阿富拉平民(Afula Civilians) : 阿富拉平民在游戏开始时遍布整个山谷，在蒙古人到来后会逃跑到地图左下方边缘，玩家必须保护他们至少1个村民存活下来才能获得胜利，在他们的基地还有几个伊玛目为玩家提供治疗

敌人
  • 蒙古伊利汗国(Mongol Ilkhanate) : 蒙古人从地图东北方向连续发动5次攻势，他们以地图西南方向的阿富拉作为目标进军，并试图击溃一切当在路上的玩家军队
  • 法兰克(Franks) : 法兰克人的十字军骑士在第三波出现，从左往右依次是步兵条顿骑士团、骑马的医院骑士团和圣殿骑士团




策略

这关是帝国时代系列历史上最难的关卡，极其考验玩家的微操和大局观，要注意多储存，尽可能减少战损，才有可能通关最高难度
玩家没有办法发展经济，只能依靠每波的援军应对敌人5波攻击
地图最南方的村落里有3个伊玛目，玩家可以让受伤的部队去接受治疗

第一波先遣部队
第一波攻击以敌人的一个单独的侦察兵为开始，后面跟随着由骑手和长矛兵组成的先遣部队
敌人的部队构成 : 12骑手 + 12骑手 + 14长矛兵 + 14长矛兵 + 14长矛兵 + 14长矛兵
玩家的英雄拜巴尔斯和12个突厥骑马弓兵会作为我方先遣队削弱敌人的先锋部队，利用突厥骑马弓兵的移动射击能力可以轻松处理掉这些部队，要注意的是，敌方骑手很可能会因为冲锋追上玩家的骑马弓兵，活用拜巴尔斯的加速技能，在敌人骑手冲锋的瞬间激活
骑马弓兵一定要尽可能地留着，它们是接下来战斗的核心

第一波中坚部队
跟随着先遣部队出现的是出现在左路的大群的骑手以及在中路出现的以民兵和步弓手组成的中坚部队，千万不要招惹中路的步弓手，玩家的骑马弓兵是完全不能处理他们的，玩家可以继续用跑打战术消耗敌人的骑手，又或是将敌人的民兵拉出来进行消耗
敌人左路：3队12骑手、中路：18民兵 + 16步弓手 + 18民兵 + 16步弓手
在敌人的中路部队到达地图中央后，玩家的援军出现：20个步弓手 + 20个长矛兵 + 20个骑手
将长矛兵部署在左路，攻击敌人的骑手，步弓手在中间攻击敌人的民兵和步弓手，骑手采取波状攻击，攻击敌人的步弓手，回避敌人的民兵，民兵可以用步弓手拉扯无伤处理掉，长矛兵在处理完骑手后后撤，不要冲上去中路送掉，骑马弓兵如果可能的话，帮助步弓手处理掉民兵

第二波
跟随在中坚部队后方的是敌人第二波部队
左路：3队20民兵、中路：2队14长矛骑兵 + 2队14蒙古突骑、右路：3队20步弓手
同时玩家的援军出现：40个武士 + 40个弩手
步弓手和骑马弓兵往左走，20个武士去右边，长矛兵、弩手和剩余的武士去中间
右边的武士主动攻击对面右路的弓箭手军团，贴着边缘攻击，减少敌人中路部队支援的可能性
控制左边的远程部队采用跑打战术吃掉对面的民兵，中间的部队让长矛兵和武士挡在前面，弩手在后面输出，偶尔控制一下武士主动攻击对方的突骑，让突骑的仇恨值放在防御力较高的武士身上，保护弩手和长矛兵

第三波
第三波是最恶心的一波，由十字军的三大骑士团成员组成，数值高，非常难打
敌人的左路：2队10弩手 + 2队12条顿骑士 + 条顿骑士团团长、中路：2队8医院骑士 + 医院骑士团团长、右路：2队8圣殿骑士 + 圣殿骑士团团长
在十字军来到地图最中央的位置后，我方援军出现：36个长矛兵 + 18个突厥骑马弓兵
弩手、武士、步弓手、长矛兵全部去右路待机。骑马弓兵列队贴着左路的边缘攻击敌人的两队弩手，同时吸引敌方条顿骑士团的注意力，敌人的条顿骑士团移动非常缓慢，不可能追得上高速移动的骑马弓兵列队，骑马弓兵将敌人的弩手和条顿骑士绕着左侧的树林跑，让他们和其他主力分开，我方其他部队理应可以以数量优势击溃敌方的两个骑士团
在圣殿骑士团和医院骑士团全军覆没后，主力部队剩余的弩手前往支援左路，采用跑打战术遛死对面所有条顿骑士团，弩手如果还有步弓手存活的话，命令剩余的步弓手处理掉

第四波
在玩家和十字军鏖战的时候，敌人的第四波部队应该已经在地图中央严阵以待
右路：3队18步弓手、中路：3队18长矛兵、左路：2队14长矛骑兵
我方援军：20个长矛骑兵 + 40个步弓手
步弓手和长矛骑兵全部前往右路，其他部队全部前往左路
长矛骑兵突击敌人右路的步弓手，注意不要和敌人右路的长矛兵硬拼，用步弓手主动攻击敌人的长矛兵军团，迫使他们将仇恨集中在步弓手身上，然后采用跑打战术减少战损
左路如果之前的战损太高的话，有可能会打不过，玩家的步弓手在快速处理掉右路的长矛兵后，火速跑到左路支援，射死敌人的长矛兵，右路可以放心将所有步弓手交给长矛骑兵

第五波
敌人的主帅怯的不花亲自出战
敌人的左路：4队10蒙古突骑、中路：2队14武士 + 3队12弩手 + 怯的不花、右路：3队12长矛骑兵
我方援军：18个骑手 + 18个武士 + 18个火枪兵 + 苏丹忽都斯
玩家的骑手、长矛骑兵和步弓手都前往左边准备应对敌人的蒙古突骑
其余部队在阿富拉的城镇中心准备战斗
敌人大概率会让所有的蒙古突骑+一队武士+一队弩手+怯的不花突击左路，其余部队突击右路。右路敌人的长矛骑兵会最先来到，在敌人的骑兵发动冲锋时，发动苏丹忽都斯的坚守技能(防御+5)，让前排部队挡住他们的攻击，火枪兵和弩手尽力输出，敌人的长矛骑兵很快就会全军覆没
左边稍微微操控制步弓手射死敌人的弩手，让骑手和长矛骑兵吸收蒙古突骑的伤害
无论结果如何，敌人应该会死剩一些落单的弩手和武士，使用剩余的部队进行最后的收尾即可

不需要刻意保存村民，因为能让敌人突破防线攻击到村民的话，已经跟输了没什么区别
村民也是分两路的，不一定要保存所有的村民，有时候专注于只保存一条路的村民更容易胜利
运气在这关占了很大的比重，在每一波开始前存档，如果战损比难以接受就果断读档吧，单位死得太多是不可能打赢后面的波次的



结尾故事

马穆鲁克动摇了蒙古人的入侵核心，所向披靡的敌人终至穷途末路。

彪悍的战马
“伟大的阿拉伯马和骑着它们驰骋战场的战士一样令人生畏。
沙漠的烈日锻造了它们的野性，寒冷的夜晚磨炼了它们的坚韧，它们是摄人心魄的尤物，世人都梦想拥有一匹。
它们有惊人的耐力和速度，长途跋涉、不吃不喝，仍能保持昂扬的精神；它们是完美的坐骑，只有最可怕的骑手配得上它们的忠诚。
当我们的公马抬起头，用力呼吸所发出的声响为法兰克人敲响他们的丧钟。
对敌人来说，它们的蹄声就像钢铁撞击沙漠的基岩，这也是敌人在下地狱前听到的人世间的最后响声。”
~ 编年史家



历史对比

  • 虽然有部分法兰克人作为蒙古人的附庸(不一定来自骑士团)，但大部分情况下十字军国家们都没有采取跟马穆鲁克苏丹国敌对的态度，甚至允许在其领地上安营扎寨以对抗蒙古人
  • 马穆鲁克苏丹国是否在这场战斗里真的使用了火铳，历史学家没有结论，但游戏里明显是因此得到灵感，在战斗中安排了一队火枪兵
