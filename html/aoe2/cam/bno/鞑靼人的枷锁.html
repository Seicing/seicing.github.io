鞑靼人的枷锁
The Tatar Yoke
剧本信息
发行版本 公爵的崛起
战役 阿基尔达斯和科斯图提斯
文明 立陶宛
鞑靼人的枷锁(The Tatar Yoke)是阿基尔达斯和科斯图提斯战役的第三个场景，基于1363年的蓝水河之战



开场故事[回到顶部]

当十字军退回去舔舐伤口的时候，很多人都以为阿基尔达斯和科斯图提斯会为了统治权火并，但他们再次失望了。兄弟俩对彼此都无比忠诚，而这也让他们得以专注于王国的扩张。

在立陶宛的东南部，是曾经的罗斯诸邦。鞑靼人的枷锁沉重地压在这些曾经繁荣昌盛的城市上。一百年来对金帐汗国的高额贡赋榨干了他们，而可汗们却在伏尔加河畔建造了金色的宫殿。

随着金帐汗国因内部纠纷而衰弱，许多被压迫的城市欢迎立陶宛来解放他们。但有些人并不热衷于只是简单地换一个统治者。

新独立的基辅城的居民认为立陶宛是更大的威胁，并寻求与他们以前的鞑靼主子结盟。当然阿基尔达斯和科斯图提斯可不会对这一举动置若罔闻。

立陶宛军队毅然南下，接连攻克切尔尼戈夫和佩列斯拉夫尔公国。这对兄弟的目标是在草原上对决金帐汗国及其盟友。这是一个大胆的举动，也是一个之前没有人敢承担的巨大风险。

尽管他们内部不和，但身经百战的鞑靼骑射手仍然是战场上不败的神话。不过阿基尔达斯和科斯图提斯决定要打破这个神话。



剧本指南[回到顶部]

开始信息
  • 时代：帝王时代
  • 资源：无
  • 人口上限：200
  • 起始单位：阿基尔达斯、13皇家骑士、14长枪兵、14弩手、2轻骑兵

主要目的
  • 通过杀死反对立陶宛统治的分离主义势力来征服基辅。
  • 科斯图提斯不得被打败。
  • 击败金帐汗国的全部三股力量 (保加利亚人、钦察人和鞑靼人)。
次要目的
  • 摧毁所有金帐汗国的船坞，终结海盗的河流封锁。
  • 杀死金帐汗国的指挥官以削弱其军队：
    • 忽格鲁特·贝格 (保加利亚人)
    • 哈吉·贝格 (钦察人)
    • 德米特里厄斯 (鞑靼人)
  • 科斯图提斯将为您收集资源。您可以通过使用以下嘲弄命令来决定他收集哪些资源：
    • 输入 3，告诉他收集更多的食物。
    • 输入 4，告诉他收集更多的木材。
    • 输入 5，告诉他收集更多的黄金。
    • 输入 6，告诉他收集更多的石料。

提示
  • 1. 阿基尔达斯的人口上限为 200 人。
  • 2. 兄弟俩商定由科斯图提斯负责经济事务，这样阿基尔达斯就可以全身心地投入到对金帐汗国的作战中。您不能训练自己的村民或建造城镇中心，科斯图提斯也不会训练很多军事单位。
  • 3. 您征服基辅之后，科斯图提斯就会立刻开始为您提供资源和人力。但您的盟友无力自保，所以当金帐汗国进攻时，您必须保护他。
  • 4. 之后，您可以通过聊天给科斯图提斯指定要收集哪些资源。使用“任务”按钮来进行这方面的管理。
  • 5. 小心从斯摩棱斯克到基辅一带徘徊的金帐汗国巡逻队。



玩家[回到顶部]

玩家
  • 阿基尔达斯(Algirdas)：玩家在斯摩棱斯克率领着一支庞大的部队，需要征服基辅后才能正常发展部队

同盟
  • 科斯图提斯(Kestutis)：科斯图提斯的基地位于基辅北部，他专注于为玩家发展经济提供必要的补给
  • 斯摩棱斯克(Smolensk)：斯摩棱斯克是玩家带着军队出发的地方，清除鞑靼海盗的河流后，玩家可以沿河道与斯摩棱斯克进行贸易

中立
  • 基辅(Kiev)：基辅已经被与金帐汗国勾结的分离主义势力夺取，正在对抗立陶宛人，阿基尔达斯必须击败由剑兵、长矛兵、和弩手组成的驻军才能征服这座城市

敌人
  • 金帐汗国(The Golden Horde)：他们是其他三个鞑靼部落的领主，在其他三个部落拥有一个坐镇的可汗，控制着基辅和斯摩棱斯克之间的巡逻部队，以及控制占领河流的鞑靼海盗
  • 保加利亚部落(The Bulgar Horde)：他们位于基辅西部，训练骑士、龙骑兵和骑射手战斗，不会使用任何攻城部队
  • 钦察部落(The Kipchak Horde)：他们在鞑靼部落的北侧拥有一个基地，训练轻骑兵和钦察骑射手进行战斗，会使用冲车进行攻城
  • 鞑靼部落(The Tatar Horde)：他们的基地在基辅的东部，训练怯薛、骑射手和草原突骑进行战斗，而且还会使用巨型投石机和冲车进行攻城




策略[回到顶部]

这个场景玩家无法自行生产村民，额外的村民只能由科斯图提斯提供，然后科斯图提斯偶尔会向玩家进贡一定的资源，即使如此也不用忘记使用村民补充资源

一开始玩家需要控制一支部队夺回地图中间的基辅，难度不大，利用兵种克制一波一波击败蓝色的驻军就可以了，不要惹怒到太多的敌人，减少受到的伤亡，击败蓝色驻军之后，玩家就会接管整个基辅，在几分钟的时间内玩家可以趁现在强化一下基辅的防御，因为敌人很快就会打来

和前两关一样，皇家骑士可以担当通杀的职责，玩家只需要无脑出这种兵种就能克制一切的敌人，当然这关骑射类的敌人也很多，如果觉得皇家骑士数量太少站不住的话，无脑出游侠也是一个可靠的选择，地图上的石头非常多，配合科斯图提斯的资源玩家可以量产一大堆皇家骑士到处乱杀，值得一提的就是，如果把缺口都围起来，科斯图提斯的基地就绝对不会受到攻击，敌人所有的攻击都会聚集到玩家控制的基辅，利用这一点把基辅构造成城堡林立的前线要塞，让敌人有去无回

三个鞑靼人的首领没有被任何城墙保护，所以用大量的骑兵直接冲到他们的城堡附近就可以轻松击杀他们的首领，大幅度削弱对应阵营的战斗力，接下来只需要按部就班把敌人逐个击破就可以了



结尾故事[回到顶部]

我们常把金帐汗国的成员称为“鞑靼人”，但实际上他们也包含很多突厥人在内。他们的军队中有挥舞梿枷的伏尔加保加利亚骑手，有来自钦察草原的库曼人，还有来自西伯利亚的蒙古人。

尽管民族和文化不同，但这些人最终还是以某种形式成为了一个整体：作为阿基尔达斯的手下败将！

当立陶宛军队向南跨过第聂伯河进入波多里亚时，金帐汗国的贝格们争先恐后地来阻击他们。

但这无济于事。立陶宛士兵手持长矛和剑，冲破了汗国的阵线。鞑靼人无法保持阵型，狼狈地撤退了。这让他们的可汗别无选择，只得逃往克里米亚。

被刀剑和箭矢杀害的马匹和战士所流下的鲜血染红了草原。阿基尔达斯决定性的胜利奠定了立陶宛大公国的大国地位，也给他带来了新的盟友和新的敌人。



杂项[回到顶部]

  • 斯摩棱斯克的桥上有一个名为米歇尔的女村民在捕鱼，这是外发人员为一个去世了的帝国时代粉丝所做的纪念



历史对比[回到顶部]

  • 这个场景基于发生在1363年的蓝水河战役，这场战斗中立陶宛大公国击败金帐汗国，把基辅和乌克兰大部分地区纳入统治之中
  • 这个场景出现的保加利亚部落是指欧亚大草原的伏尔加保加尔人，而不是巴尔干半岛上的保加利亚王国，伏尔加保加尔是一个独立的汗国，存在于7到13世纪，在本场景发生的时代他们基本上被金帐汗国同化了
  • 场景里的钦察部落也指代了13世纪末被蒙古人击败并且同化的前库曼-钦察联盟的成员
